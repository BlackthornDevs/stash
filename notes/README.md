# Notes

This directory contains notes, brain dumps, and quick thoughts on various topics.

## Format

Notes can be in any format that works for you:
- Markdown files (.md)
- Text files (.txt)
- Code files for technical notes

## Suggested Structure

```markdown
# [Topic/Title]

**Date**: YYYY-MM-DD
**Tags**: #tag1 #tag2

## Content

Your notes here...

## Follow-up

- Action items
- Questions
- Related topics to explore
```

## Organization

Organize notes however makes sense - by date, topic, project, or in a single file.
