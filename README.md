General purpose repo for brain dumps and random ideas.

## Purpose

This repository serves as a personal knowledge base and idea collection. It's a place to capture:
- Quick ideas and project concepts
- Notes from learning and research
- Useful code snippets and scripts
- Curated resources and references

## Structure

```
stash/
├── ideas/          # Project ideas and concepts
├── notes/          # Notes and brain dumps
├── code-snippets/  # Useful code snippets and scripts
└── resources/      # Curated lists and references
```

## Usage

### Adding New Content

1. **Ideas**: Create a new markdown file in `ideas/` using the template in `ideas/README.md`
2. **Notes**: Add notes in `notes/` - can be quick thoughts or detailed documentation
3. **Code Snippets**: Save reusable code in `code-snippets/`, organized by language or purpose
4. **Resources**: Maintain curated lists in `resources/`, organized by topic

### Organization Tips

- Use descriptive filenames
- Add dates and tags to your entries
- Create subdirectories for better organization
- Link between related items
- Keep examples for reference (feel free to delete the example files when you're ready)

## Getting Started

Each directory contains a README with format suggestions and examples. Feel free to adapt the structure to your needs!
