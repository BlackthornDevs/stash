# Resources

This directory contains curated lists of resources, links, tools, and references.

## Format

Resources can be organized as:
- Markdown files with lists of links
- Topic-specific collections
- Tool recommendations
- Learning resources

## Suggested Structure

```markdown
# [Resource Category]

**Last Updated**: YYYY-MM-DD

## [Subcategory]

- [Resource Name](URL) - Brief description
- [Another Resource](URL) - Brief description

## Notes

Additional context or recommendations
```

## Organization

Create separate files for different topics or maintain a single master list.
