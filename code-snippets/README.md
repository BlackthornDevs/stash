# Code Snippets

This directory contains useful code snippets, scripts, and small utilities.

## Organization

- Organize by language or purpose
- Include comments explaining what each snippet does
- Add usage examples where applicable

## File Naming

Use descriptive names:
- `python/fetch-api-data.py`
- `javascript/debounce-function.js`
- `bash/git-cleanup.sh`

## Template

```
# [Snippet Name]

## Description
What this snippet does

## Usage
How to use it

## Code
The actual code with comments
```
